# Hospital Patient Management System

A Python mini project developed using Object-Oriented Programming concepts for managing hospital patient records.

## Features

* Add Patient
* Display All Patients
* Search Patient by ID
* Delete Patient
* Save Patient Data to File
* Exception Handling
* Menu-Driven Interface

## Concepts Used

* Variables
* If-Else Statements
* Loops
* Functions
* Lists
* Classes and Objects
* File Handling
* Exception Handling
* Object-Oriented Programming

## Patient Information Stored

* Patient ID
* Name
* Age
* Disease
* Blood Group

* hospital_management.py
* patients.txt
* README.md
